# Career Repositioning con IA

Un prompt estructurado para reposicionarte profesionalmente hacia cualquier rol técnico emergente — usando IA como estratega de carrera.

Diseñado para profesionales tech de LATAM que apuntan a roles remotos en empresas globales.

---

## Cómo usarlo

1. Abrí [`career-reposition-prompt.md`](career-reposition-prompt.md)
2. Copiá el prompt completo
3. Pegalo en una conversación nueva con Claude, ChatGPT, o el LLM que uses
4. Completá las secciones entre `[corchetes]` con tu información real:
   - El rol al que querés llegar (cualquier rol tech)
   - Tu perfil actual
   - Tu presupuesto y disponibilidad
5. Ejecutá las 4 fases en orden — no saltes ninguna

> Cuanto más honesto seas con tu perfil actual, más útil va a ser el resultado.

---

## El prompt en 4 fases

```
Fase 1 → Job Description del rol objetivo (con datos de mercado)
Fase 2 → Gap Analysis de tu perfil actual vs. ese rol
Fase 3 → Roadmap de aprendizaje y certificaciones
Fase 4 → Plan de estudio detallado para los próximos 2-3 meses
```

Cada fase se construye sobre la anterior. El resultado es un plan ejecutable, no motivación genérica.

---

## Para quién es esto

- Profesionales tech con experiencia que quieren hacer una transición de rol
- Developers, architects, consultores, o analistas que quieren reposicionarse
- Cualquiera que prefiera un diagnóstico concreto por encima de un curso genérico

El prompt funciona para cualquier rol: AI Engineer, DevOps, Platform Engineer, Data Engineer, Product Engineer, Cloud Architect, o el que tengas en mente.
